git распределенная система контроля версии
Это система для отслеживание и ведение истории изменении файлов, в проекте
Репозиторий - хранилише вашего кода и истории его изменений
GitHab  - Веб сайт для размишение git-репозиториев в совмесной разработке проектов

Команды  GIT :
git init- она создает новый локальный гит репозиторий у на пк, в той директории в котором
была прописана команда.
git add -команда для добавления файлов в систему отслеживание гита , после это команды изменения начинают отслеживатся гитом.
git commit -m '<comment>'- как только достигаем определенного момента в разработке то тогда мы сохранаем и комментируем изменения в лакальный репазитории
git remote add = это команда для того чтобы связывать наш локальный репозиторий с удаленным репозиторием то есть добавляет подключение на гитхабе
git push- команда для отправки всех ваших изменений на локальном репоз-и в удаленный репоз-и
git push origin main
