linux comands
ctrl+alt+t комонда для открытия 
~ символ домашней директории
/ синвол корневей директрии
.. обозначает предведущеу директории
- обозначает текущую директории
1.ls команда вывода всех директории и файлов
(ls -a) вывод скрытых директории
cd для переходов по директрии cd <kuda>
mkdir создание директории (папки)
touch создает файл
rm  удалить файл
rmdir удаляет директории (папки)
rm -rf удалить принудительно 
pwd отображает полный путь до текушей директории
nano чтобы редактировать техтывые файлы
cat для просмотра без изменения
echo для быстрой записи в файл (echo 'текст' > название файла) 
mv команда для переименования файла а так же для перемишения файлов
